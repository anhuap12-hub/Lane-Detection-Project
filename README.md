# Advanced Lane Detection System | YOLOv5 & YOLOv8

This project implements and compares state-of-the-art Object Detection models (YOLOv5 and YOLOv8) specifically for the **Lane Detection** task.

## 🚀 Overview
The goal is to develop a robust system capable of identifying and tracking road lane lines in real-time, which is a fundamental component for autonomous driving and advanced driver-assistance systems (ADAS).

## 🛠️ Key Features
- **Dual Model Implementation:** Hands-on experience with both YOLOv5 (Ultralytics) and the latest YOLOv8.
- **Custom Workflow:** Data preprocessing, model training, and performance benchmarking.
- **Performance Analysis:** Comparison of inference speed (FPS) and accuracy (mAP) between versions.
- **Visualization:** Integrated OpenCV for real-time video processing and result overlay.

## 💻 Tech Stack
- **Languages:** Python
- **Frameworks:** PyTorch, Ultralytics YOLO
- **Libraries:** OpenCV, NumPy, Matplotlib, Pandas
- **Environment:** Google Colab (GPU accelerated)

## 📂 Project Structure
- `Lane_Detection_yolov5.ipynb`: Implementation using YOLOv5.
- `Lane_detection_yolov8.ipynb`: Advanced implementation using YOLOv8.
